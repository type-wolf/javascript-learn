## Learn JavaScript & TypeScript

### Execution

```bash
npm start {category} {dir} {fileName}
```

-   #### `category` : The category of the file you want to run
-   #### `dir` : Specify the directory after `src
-   #### `fileName` : Specify the name of the file you want to run
