## Learn JavaScript & TypeScript Mechanism

### Execution

```bash
npm start {dir} {fileName}
```

-   #### `dir` : Specify the directory after `src
-   #### `fileName` : Specify the name of the file you want to run

### Sample

```txt
├─ .git
├─ node_modules/
├─ dist/
├─ src/
│  ├── variable/
│  │   └─ type.ts
│  │   └─ README.md
│  └─ index.ts
├─ .eslintrc.js
├─ .env
├─ .gitignore
├─ .prettierrc
├─ .README.md
├─ package-lock.json
├─ package.json
└─ tsconfig.json
```

```bash
npm start variable type
```
